## Description

ZedMark is a native macOS Markdown reader built for people who want a clean place to read and review local Markdown files.

Open Markdown or plain text documents, browse headings in the outline, switch between reader themes, and keep the preview updated as the source file changes. ZedMark also includes document stats, lightweight proofing checks, recent files, and export to HTML or PDF.

Highlights:

- Open `.md`, `.markdown`, `.mdown`, and plain text files
- Drag and drop files into the reader
- Live preview reloads when the source file changes
- Sidebar with recent files and heading outline
- Reader themes for different writing and reading styles
- Word, line, heading, and reading-time stats
- Lightweight proofing for repeated words, long sentences, and TODO markers
- Export the rendered preview to HTML or PDF
- Open the source file in your editor or reveal it in Finder
- English and Simplified Chinese interface

ZedMark keeps the workflow simple: open a local Markdown file, read it in a polished preview, review its structure, and export when needed.
