# ZedMark Demo Note

ZedMark is a native Markdown reader for macOS. Use it to open a local note, review structure, preview styled Markdown, and export a polished copy to HTML or PDF.

## Read Local Markdown

Open `.md`, `.markdown`, `.mdown`, and plain text files from Finder, the File menu, or drag and drop. ZedMark keeps the document local and reloads the preview when the source file changes on disk.

## Navigate With Outline

The sidebar shows recent files and a heading outline, so long notes stay easy to scan.

### Quick Review

- Word, line, heading, and reading-time stats
- Lightweight proofing for repeated repeated words
- TODO markers are surfaced in the inspector
- Live preview for source changes

## Pick A Reading Style

Choose from reader themes such as Claude, GitHub, Notion, Paper, Solarized, Nord, Catppuccin, Academic, Carbon, and Mono.

> A quiet Markdown reading workspace for notes, drafts, specs, and documentation.

## Export

When the preview looks right, export the rendered document to HTML or PDF.

| Workflow | Result |
| --- | --- |
| Open Markdown | Styled preview |
| Review outline | Fast navigation |
| Export HTML/PDF | Shareable output |
