# ZedMark Support

ZedMark is a native macOS Markdown reader for local Markdown and plain text files.

## Getting Help

For support, bug reports, or feature requests, use the GitHub issues page:

[https://github.com/ieaglez/ZedMark/issues](https://github.com/ieaglez/ZedMark/issues)

## Basic Troubleshooting

- Make sure the file is a Markdown or plain text file.
- If a recent file cannot be reopened, use **Open Markdown** and select the file again.
- If export does not complete, try exporting to a different folder where you have write permission.
